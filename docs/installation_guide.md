
# Inštalacija

## 1. Nastavitev backenda
Odpri PowerShell:
```bash
cd backend
pip install -r requirements.txt
python app.py
```
Backend dela na:
http://127.0.0.1:5001

## 2. Desktop Client
```bash
cd desktop_client
python client.py
```
## 3. Web Client
Odpri:
```text
web_client/index.html
```
v kateremkoli brskalniku (priporočeno: Chrome).
