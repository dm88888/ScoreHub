# ScoreHub — Načrt razvoja

##  Zaključeno (MVP)
- Backend API (Flask)
- Upravljanje igralcev
- XP/Level/Score sistem
- Dnevna nagrada
- Heroji (5)
- Sistem bitk (turn-based)
- Namizni odjemalec (Tkinter)

- Osnovna dokumentacija
